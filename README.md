# Neos-asteroides
NEOS v1 — Ferramenta em Python para monitoramento de asteroides próximos da Terra, ideal para estudos astronômicos, automação e ciência de dados.

# ☄️ NEOS v1 — Monitor de Asteroides Próximos da Terra

Projeto desenvolvido por **Cleytson Miguel de Oliveira**.

## 🔭 Sobre o projeto
O NEOS v1 é um script em Python que consulta dados de asteroides próximos da Terra (Near-Earth Objects) e exibe:

- Nome do asteroide
- Distância da Terra
- Velocidade
- Potencial risco
- Informações científicas relevantes

## 🚀 Aplicações
- Estudos de astronomia
- Ciência de dados espaciais
- Automação com Termux
- Monitoramento de objetos celestes

## 🛠️ Tecnologias
- Python
- APIs astronômicas
- Automação via Termux

```bash
python neos_v1.py
